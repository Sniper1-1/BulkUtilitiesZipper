<h1> Bulk Utilities Zipper </h1>

This Python tool was designed for personal use to quickly select multiple folders and create a respective .zip file for their contents, but if you want to use B.U.Z. for your own use case, feel free.


<h2>Usage</h2>

Run the .py file and select your first folder you intend to zip. Continue selecting folders and hit the cancel button once done. This will prompt you to start selecting the directories to save the .zip files to.
